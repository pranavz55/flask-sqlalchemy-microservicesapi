# flask-sqlalchemy-microservicesapi  
This is microserviceapi using Flask SqlAlchemy and Marshmallow  
Validation is done using Postman API  
# Features:  
1.Create Post  
2.View all Posts/ posts by Id   
3.Delete Post by Id  
4.Update particular post by Id
